# **App Name**: NC Properties Redefined

## Core Features:

- Homepage Hero Section: Display a hero section with an overlay text to introduce the property listings.
- Advanced Search Box: Implement a prominent search box for users to quickly find properties based on specified criteria.
- Property Swiper Carousel: Use a Swiper component to showcase property cards with key details and attractive visuals.
- Categorized Property Browsing: Allow users to explore properties via CTA (Call to Action) tabs, making navigation intuitive and efficient.
- Newsletter Subscription: Offer users the opportunity to subscribe to a newsletter, encouraging continued engagement with the NC Properties.
- Company Information: Detail NC Properties’ background, mission, and team using clear timelines and employee profiles.
- Support Form: Provide a contact form for support questions.

## Style Guidelines:

- Primary color: Deep maroon (#4B0000) to represent sophistication and tradition.
- Background color: Whitesmoke (#F5F5F5), providing a clean and neutral backdrop.
- Accent color: Burgundy (#BB0000), analogous to deep maroon, to add a subtle contrast without overshadowing.
- Headline font: 'Roboto' (sans-serif) for modern, readable titles.
- Body font: 'Lato' (sans-serif) providing a clean and accessible reading experience. Note: currently only Google Fonts are supported.
- Utilise Font Awesome icons (e.g., fa-heart, fa-user, fa-magnifying-glass) for interactive elements.
- Implement a responsive layout using CSS Grid and Flexbox with a max-width of 1100px on desktop and 90% on smaller screens to ensure consistent rendering.
